# mission.md

## Projektname

Sitzplatzbuchungssystem für eine Krankenkasse

## Mission

Ziel des Projekts ist die Entwicklung eines webbasierten Sitzplatzbuchungssystems für eine Krankenkasse mit den Standorten Ingolstadt, München und Wolfsburg. Das System soll Mitarbeitern ermöglichen, freie Sitzplätze an einem gewünschten Standort für einen bestimmten Arbeitstag oder Zeitraum einfach zu suchen und zu buchen.

## Ausgangssituation

Die Krankenkasse besitzt drei große Standorte. Der Standort Ingolstadt besteht aus drei Stockwerken. Die Standorte München und Wolfsburg besitzen jeweils nur ein Stockwerk. Mitarbeitende benötigen eine einfache Möglichkeit, vorab einen passenden Arbeitsplatz zu reservieren.

Ohne ein zentrales Buchungssystem kann es zu Doppelbuchungen, unklarer Sitzplatzverteilung und erhöhtem organisatorischem Aufwand kommen.

## Zielgruppe

Die Hauptzielgruppe des Systems sind Mitarbeitende der Krankenkasse.

## Nutzen

Das System bietet folgende Vorteile:

- Mitarbeitende können freie Sitzplätze einfach finden und buchen.
- Doppelbuchungen werden verhindert.
- Die Standort- und Stockwerksstruktur wird berücksichtigt.
- Mitarbeitende können eigene Buchungen einsehen und stornieren.
- Administratoren erhalten eine bessere Übersicht über die Sitzplatznutzung.
- Datenschutzanforderungen können durch rollenbasierte Zugriffsrechte berücksichtigt werden.

## Abgrenzung

Das System dient ausschließlich der Buchung von Sitzplätzen innerhalb der Krankenkasse. Es ersetzt keine allgemeine Raumverwaltung, keine Urlaubsplanung und keine Personalplanung.

Nicht Bestandteil der ersten Version sind:

- Buchung von Besprechungsräumen
- Parkplatzreservierungen
- automatische Auswertung der Anwesenheit
- Integration in externe Gebäudemanagementsysteme
- mobile App als eigenständige Anwendung

## Erfolgskriterien

Das Projekt gilt als erfolgreich, wenn Mitarbeitende einen freien Sitzplatz suchen, buchen, einsehen und stornieren können. Außerdem muss das System verhindern, dass ein Sitzplatz im gleichen Zeitraum mehrfach gebucht wird.
